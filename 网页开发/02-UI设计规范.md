## 设计理念

### 核心原则

- **专注学习**：减少视觉干扰，突出内容本身
- **舒适阅读**：优化长文阅读体验，降低视觉疲劳
- **高效导航**：清晰的层级结构，快速定位内容
- **现代专业**：简洁大气的设计语言，体现专业品质

### 设计关键词

`专注` `清晰` `高效` `现代` `专业`

## 整体布局设计

### 页面结构

```
┌──────────────────────────────────────────────────────────────┐
│                        顶部导航栏 (64px)                        │
├─────────────┬────────────────────────────────────────────────┤
│             │                                                │
│   目录栏     │              内容区域                           │
│   (250px)   │           (Markdown渲染)                        │
│   可折叠     │                                                │
│             │                                                │
│             │                                                │
│             │                                                │
└─────────────┴────────────────────────────────────────────────┘
```

### 布局参数

| 区域 | 宽度 | 高度 | 说明 |
|------|------|------|------|
| **顶部导航栏** | 100% | 64px | 固定顶部，包含Logo、搜索、用户信息 |
| **目录栏** | 250px | calc(100vh - 64px) | 可折叠，默认展开 |
| **内容区** | calc(100% - 250px) | calc(100vh - 64px) | 主内容区，可滚动 |
| **内容最大宽度** | 900px | - | 内容区最大宽度，居中显示 |

### 目录栏设计

```
┌─────────────────────┐
│ 📚 课程目录     [◀] │  ← 标题栏 + 折叠按钮
├─────────────────────┤
│ 🔍 搜索课程...      │  ← 搜索框 (可选)
├─────────────────────┤
│ ▼ 第一章 简介       │  ← 一级标题
│   ├ 1.1 什么是AI    │  ← 二级标题
│   └ 1.2 AI的历史    │
│ ▶ 第二章 基础概念   │  ← 折叠状态
│ ▶ 第三章 ...        │
└─────────────────────┘
```

### 目录栏状态

- **展开状态**：宽度 250px
- **折叠状态**：宽度 0px，显示展开按钮
- **响应式**：平板设备默认折叠

```css
/* 移动端优先 */

/* 小屏手机 */
@media (max-width: 639px) {
  .sidebar { display: none; }
  .content { padding: 16px; }
}

/* 大屏手机/平板竖屏 */
@media (min-width: 640px) and (max-width: 767px) {
  .sidebar { display: none; }
  .content { padding: 24px; }

  }
}

/* 桌面端 */

  .sidebar { width: 250px; }

    max-width: 900px;
    padding: 48px;
    margin: 0 auto;
  }
}

/* 大屏 */

}
```

## 动效规范

### 过渡动画

```css
/* 基础过渡 */
.transition-base {
  transition: all var(--duration-fast) var(--ease-in-out);
}

/* 颜色过渡 */
.transition-colors {
  transition: color, background-color, border-color var(--duration-fast) var(--ease-in-out);
}

/* 变换过渡 */
.transition-transform {
  transition: transform var(--duration-normal) var(--ease-in-out);
}

/* 阴影过渡 */
.transition-shadow {
  transition: box-shadow var(--duration-normal) var(--ease-in-out);
}
```

### 目录栏折叠动画

```css
/* 目录栏展开/折叠动画 */
.sidebar {
  transition: transform var(--duration-normal) var(--ease-in-out),
              width var(--duration-normal) var(--ease-in-out);
}

.sidebar.collapsed {
  transform: translateX(-100%);
  width: 0;
}

/* 内容区自适应 */
.content {
  transition: margin-left var(--duration-normal) var(--ease-in-out);
}

.sidebar.collapsed + .content {
  margin-left: 0;
}
```

### 悬停效果

```css
/* 卡片悬停 */
.card-hover {
  transition: transform var(--duration-normal) var(--ease-out),
              box-shadow var(--duration-normal) var(--ease-out);
}

.card-hover:hover {
  transform: translateY(-2px);
  box-shadow: var(--shadow-lg);
}

/* 按钮悬停 */
.btn-hover {
  transition: background-color var(--duration-fast) var(--ease-in-out),
              transform var(--duration-fast) var(--ease-in-out);
}

.btn-hover:hover {
  transform: translateY(-1px);
}

.btn-hover:active {
  transform: translateY(0);
}
```
