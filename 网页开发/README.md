# AI大模型技术与商业应用 - 在线学习课程网站

## 📖 项目简介

这是一个基于现代Web技术构建的在线课程学习平台，用于展示和阅读《AI大模型技术与商业应用》课程的41个章节内容。采用响应式设计，支持桌面端和移动端访问。

## ✨ 功能特性

### 核心功能
- ✅ **左侧章节导航** - 41章内容按八大模块分组展示
- ✅ **右侧内容展示** - Markdown格式渲染，支持代码高亮
- ✅ **章节切换** - 点击目录即可快速切换章节
- ✅ **目录折叠/展开** - 模块可独立折叠，支持全部展开/收起
- ✅ **响应式布局** - 完美适配桌面、平板和手机
- ✅ **阅读进度** - 顶部进度条实时显示阅读进度
- ✅ **章节导航** - 文章底部支持上一章/下一章快速跳转

### 界面特点
- 🎨 **现代深色主题** - 护眼舒适的学习环境
- 🚀 **流畅动画效果** - 精心设计的交互反馈
- 📱 **移动端优化** - 汉堡菜单、触摸友好
- 🖨️ **打印友好** - 支持打印章节内容

## 🛠️ 技术栈

### 前端技术
- **HTML5** - 语义化标签
- **CSS3** - Grid/Flexbox布局、CSS变量、动画
- **JavaScript (ES6+)** - 原生JS，无框架依赖

### 第三方库
- **Marked.js v12.0.0** - Markdown解析和渲染
- **Highlight.js v11.9.0** - 代码语法高亮

## 🚀 快速开始

### 方式一：直接打开（推荐用于本地学习）

1. 双击 `index.html` 文件
2. 浏览器会自动打开课程网站
3. 点击左侧目录开始学习

### 方式二：本地服务器（推荐用于开发调试）

由于浏览器安全限制，使用 `fetch` 加载本地文件需要启动本地服务器。

#### 使用 Python 启动：
```bash
# Python 3
cd "C:\Users\Administrator\.qclaw\workspace\AI课程\网页开发"
python -m http.server 8000

# 然后访问 http://localhost:8000
```

#### 使用 Node.js 启动：
```bash
# 安装 http-server
npm install -g http-server

# 启动服务器
cd "C:\Users\Administrator\.qclaw\workspace\AI课程\网页开发"
http-server -p 8000

# 访问 http://localhost:8000
```

#### 使用 VS Code Live Server：
1. 安装 Live Server 扩展
2. 右键 `index.html` 选择 "Open with Live Server"

## 🎯 使用指南

### 基本操作
1. **选择章节** - 点击左侧目录的章节标题
2. **展开模块** - 点击模块名称展开/收起章节列表
3. **章节导航** - 使用文章底部的"上一章/下一章"按钮
4. **阅读进度** - 查看顶部进度条了解当前位置

### 快捷操作
- **全部展开** - 点击"全部展开"按钮展开所有模块
- **移动端菜单** - 点击右下角菜单按钮打开目录

### URL导航
支持直接通过URL访问特定章节：
```
index.html#chapter-1   # 直接打开第1章
index.html#chapter-25  # 直接打开第25章
```

## ⚙️ 自定义配置

### 修改课程路径
在 `script.js` 中修改 `CONFIG.coursePath`：
```javascript
const CONFIG = {
    coursePath: '../',  // Markdown文件相对路径
    // ...
};
```

### 修改章节文件映射
在 `CONFIG.chapterFiles` 中添加或修改章节：
```javascript
chapterFiles: {
    1: '01-开学典礼.md',
    // 添加新章节...
}
```

### 自定义主题颜色
在 `styles.css` 中修改 CSS 变量：
```css
:root {
    --primary-color: #667eea;     /* 主色调 */
    --secondary-color: #764ba2;   /* 辅助色 */
    --bg-primary: #0f0f23;        /* 主背景色 */
    --bg-secondary: #1a1a2e;      /* 次背景色 */
    /* ... */
}
```

## 🔧 故障排除

### 问题1：章节内容无法加载
**原因**：浏览器安全策略限制本地文件访问

**解决方案**：
- 使用本地服务器（推荐）
- 或使用 Chrome 参数：`--allow-file-access-from-files`

### 问题2：样式显示异常
**解决方案**：
- 清除浏览器缓存（Ctrl+F5）
- 检查 CSS 文件路径是否正确

### 问题3：移动端菜单不响应
**解决方案**：
- 检查 JavaScript 是否正常加载
- 查看浏览器控制台错误信息
