## 1. 产品概述

### 1.1 产品背景

《AI大模型技术与商业应用》课程包含41章专业AI学习内容，涵盖从AI基础认知到企业级部署应用的完整知识体系。目前课程内容以Markdown格式存储，缺乏统一的展示和阅读平台，用户难以便捷地浏览和学习。

本产品旨在构建一个专业的课程学习网站，为用户提供清晰、流畅的课程阅读体验。

### 1.2 产品定位

**面向AI学习者的在线课程阅读平台**

- 目标用户：技术管理者、AI开发者、企业数字化转型团队
- 核心价值：提供结构化的AI课程内容，支持便捷的章节导航和内容阅读
- 产品形态：静态网页应用，支持本地运行

### 1.3 课程内容概览

| 模块 | 章节范围 | 内容主题 |
|------|----------|----------|
| 基础认知与趋势 | 第1章 | 课程导览、AI发展趋势 |
| 提示工程与对话 | 第2章 | Prompt设计方法论 |
| RAG检索增强 | 第8-13章 | 向量数据库、知识库构建、RAG优化 |
| Agent智能体 | 第3-4、14-19章 | Agent架构、工具开发、多Agent协作 |
| AI框架与开发 | 第20-25章 | LangChain、PyTorch、HuggingFace |
| 模型微调 | 第26-31章 | LoRA、数据工程、蒸馏技术 |
| 企业级部署 | 第32-34章 | 硬件选型、性能优化、SGLang |
| 低代码平台 | 第35-36章 | Coze、Dify应用实践 |
| 高级应用 | 第37-41章 | Agent运维、测试、ChatBI |

## 2. 用户故事

### 2.1 核心用户画像

**用户画像 A：技术管理者**
- 背景：企业技术负责人，需要快速了解AI技术全貌
- 需求：系统性地了解AI技术栈，为团队技术选型提供依据
- 痛点：内容分散，缺乏系统性学习路径

**用户画像 B：AI开发者**
- 背景：正在学习或实践AI应用开发
- 需求：深入特定章节，查阅技术细节
- 痛点：需要在多个章节间跳转，对比学习

**用户画像 C：课程学员**
- 背景：正在系统学习本课程
- 需求：按顺序阅读章节，记录学习进度
- 痛点：需要清晰的章节导航和阅读位置记录
